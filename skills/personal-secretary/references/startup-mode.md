# Startup Mode

Enter startup mode when critical initialization files are missing.

Critical files:
- `data/profile/personal-profile.md`
- `data/profile/startup-config.md`
- `data/memory/long-term-memory.md`
- `data/notion/notion-config.md`

Startup stages:
1. Build the personal profile
2. Define or confirm the main domain framework
3. Configure greeting and review rhythm
4. Initialize local memory files
5. Initialize Notion three-database mirroring

Startup is not complete until those files exist and contain real initial content.

If the user later asks to revise setup, enter startup revision mode and change only the relevant files.
